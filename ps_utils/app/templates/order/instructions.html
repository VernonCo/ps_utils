Instructions for POSTing to Inventory, Order Status, and Shipping Notification Forms

POST's for these forms require form_encoded data!

Data submitted should follow the WSDL schema, field names, and their types. Grab POST data from the manual entry forms for examples.
Endpoints:

  • Inventory: /inventory/index/
  • Order Status: /orderstatus/index/
  • Inventory: /shippingstatus/index/

Future plans enclude accepting json requests and validation data before sending the SOAP request similar to the purchase order.
